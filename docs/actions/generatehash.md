
## Generate Hash / generatehash (internally is.workflow.actions.hash)


### usage
`generatehash type=[string <${strInfo}>]`

### arguments
### Enumeration: Type / type (internally WFHashType)
**Default Value**: MD5
**Allows Variables**: true


Accepts a string 
or variable
containing one of the options:

- `MD5`
- `SHA1`
- `SHA256`
- `SHA512`
